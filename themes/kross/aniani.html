AniAni

AniAni React Project

github || aniani.herokuapp.com

AniAni is a solo project I created using React and express. It uses Jikan api to pull data from an anime database. This site allows the user to search their favorite anime and view information about it . Such as air date, synopsis, and a trailer. If the user is logged in they can also add anime to favorites list. Other technologies used to make this are redux, PostgreSQL, and axios.
